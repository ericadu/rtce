README: How to Run Editr

1. Open main.package
2. Open command line. To run Server.java file, cd into directory where file is stored, then type: 
	Server.java PORT_NUM
	where PORT_NUM is your desired port number
3. Run Client.java as a Java Application
4. Enter in a username, server (i.e. hostname or ip address, like "localhost"), and a port (number)

Enjoy! Thank you for trying Editr.
